## Project overview

This repository quantifies the welfare cost of tax rate misperception. Workers who misperceive their marginal tax rate choose suboptimal labor supply, generating deadweight loss. The core result: E[DWL]/earnings ≈ ½εσ²/(1-τ).

## Architecture

- `src/taxuncertainty/` - Python package with models, analysis, and pipeline
- `tests/` - pytest test suite (66 tests, 99% coverage)
- `paper/` - MyST-based academic paper
- `src/taxuncertainty/data/results.json` - All computed results (generated by pipeline.py)

## Development

- Python 3.13+, pytest for testing
- `pip install -e ".[dev]"` to install
- `python -m pytest tests/` to run tests
- All results are deterministic (seed=42)

## Key model

Quasilinear-isoelastic preferences: U(C,h) = C - ψh^(1+1/ε)/(1+1/ε)
Optimal hours: h* = (w(1-τ)/ψ)^ε
DWL from misperception: E[DWL] ≈ ½εwh*σ²/(1-τ)
